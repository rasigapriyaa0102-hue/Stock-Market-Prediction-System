About This Project

This project focuses on forecasting future stock prices using a Hybrid Deep Learning approach. The system combines Long Short-Term Memory (LSTM) and Gated Recurrent Unit (GRU) models to effectively learn both past and future dependencies in time-series stock data.

The model is trained on historical stock prices and predicts the next 30 days of future values. The results are visualized using charts and tables, helping users understand potential price trends.

This system does not provide financial decisions such as Buy or Sell recommendations, but focuses purely on accurate forecasting using deep learning techniques.
